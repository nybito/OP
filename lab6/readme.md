# Задание:
Геометрические тела

    Параллелепипед
    Тетраэдр
    Шар

Расчёт объема, площади поверхности, массы в зависимости от материала.

# Описание работы:
Есть основной файл main, в котором происходит создание окна через tkinter и сбор информации через entry.get(). 
Функция calculate обращается к функциям внутри пакета packet и получает результаты вычислений. 
Функция save_report сохраняет результаты в doc и xls.
Функция update_fields обновляет лейблы внутри окна, для выключения тех окон, информация с которых не нужна.
